# Global Economic Indicators Analysis

This project analyzes global macroeconomic data across multiple countries and years.

The notebook includes data cleaning, exploratory data analysis, visualization of economic trends, and machine learning techniques to explore relationships between indicators such as GDP growth, trade openness, inflation, unemployment, foreign investment, and oil prices.

It also applies clustering to group countries with similar economic patterns.

Tools used: Python, Pandas, Matplotlib, Seaborn, Scikit-learn.
